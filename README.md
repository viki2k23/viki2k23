# Hi, I'm Vicky 👋

**Remote Operations Professional | Property Management | Systems Builder**

I help U.S.-based property managers and business owners run cleaner,
more efficient operations from a distance. I specialize in building
repeatable systems, handling daily operational work, and documenting
processes that keep portfolios and businesses running consistently.
